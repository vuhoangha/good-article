    Tài liệu machine learning cơ bản là cái cần thiết nhất đối với các bạn mới bắt đầu với machine learning. Nhưng những bạn mới bắt đầu học sẽ khó khăn hơn trong việc tìm kiếm các tài liệu hay và phù hợp. Vì vậy, mình muốn tạo ra topic này để tổng hợp các tài liệu mà mình được tiếp cận và mình cho là tốt cho những bạn mới bắt đầu học.

    Mình sẽ cố gắng tổng hợp lại topic này thành 1 trang wiki. Vì vậy, rất mong các bạn nếu có tài liệu nào hay hãy chia sẻ cho tất cả mọi người bằng cách để lại nó ở mục bình luận.

    Danh sách dưới đây mình sẽ bổ sung dần. Mình cũng rất mong các bạn có thể bổ sung giúp mình xuống mục bình luận các tài liệu mà bạn thấy hay.

    Thông báo:

    Mình sẽ liên tục cập nhật các tài liệu machine learning cơ bản hay tại bài viết này. Các bạn nên bookmark lại trên trình duyệt để tiện theo dõi.

    Tuy nhiên, trước khi chia sẻ tài liệu mình xin chia sẻ cách để các bạn tự tìm kiếm tài liệu trên trang github. Đây là một kho tài liệu vô giá được cả một cộng đồng xây dựng. Hi vọng các bạn có thể tiếp cận được với kho tài liệu này.

    Tìm kiếm tài liệu machine learning cơ bản trên github

    Github là một kho tài liệu lập trình đồ sộ. Và tất nhiên, nếu bạn muốn tìm tài liệu tự học machine learning cơ bản thì đây là kho tài liệu tuyệt vời cho bạn. Bây giờ tôi sẽ chỉ các bạn cách tìm kiếm tài liệu trên này.

    Tìm kiếm theo từ khóa

    B1. Truy cập trang github.com
    B2. Tìm kiếm theo từ khóa awesome-xxx. Trong đó xxx là cái bạn muốn tìm kiếm. Xem ảnh để rõ hơn.

    Từ “awesome” là cái gì đó rất tuyệt vời, rất đặc biệt.

    B3. Xem và chọn repo có nhiều star nhất thôi.

    Với cách tìm kiếm này, bạn có thể tìm thấy hầu hết mọi tài liệu machine learning cơ bản awesome mà bạn cần. Tất nhiên, bạn cũng có thể tìm theo từ khóa bất kỳ nhé(VD: pandas-tutorial, python-tutoral,…).

    Lưu ý:

    1. Bạn nên thay dấu cách bằng dấu ‘-’. Vì tên project trên này thường không để dấu cách.
    2. Đừng quên ta có Google search nữa nhé.

    Theo dõi trending repos trên github

    Do github là một cộng đồng lớn nên mỗi ngày có rất nhiều repo được tạo ra. Nhưng không phải repo nào cũng hữu ích với bạn. Có một cách đơn giản để bạn nhanh chóng tìm ra repo hữu ích và có nhiều người quan tâm.

    Để biết repo nào đang có nhiều người quan tâm(trendding). Hãy truy cập trang Github trending này để xem danh sách các repo được nhiều người star(giống like trên FB) nhất theo ngày/ tuần/ tháng nhé.

    Bạn cũng nên truy cập vào trang explore để khám phá các repo theo topic, mức độ phổ biến, cũng như các repo mà github gợi ý theo sở thích của bạn.

    Đường dẫn tới github trending: https://github.com/trending
    Đường dẫn tới trang khám phá: https://github.com/explore

    Một số repos hay ho trên github

    Phần này mình sẽ chia sẻ cho các bạn những repo hữu ích nhất, awesome nhất cho người mới học machine learning.

    1. Lộ trình học Machine Learning, Deep Learning cho người mới bắt đầu

    Repo này chia sẻ cho bạn những kiến thức cần thiết để học tốt Machine Learning trong vòng 2 tháng.

    https://github.com/bangoc123/learn-machine-learning-in-two-months

    2. Tổng hợp tất cả các tài liệu hay về machine learning

    Đây là repo tổng hợp tất tần tật những tài liệu awesome về machine learning.

    https://github.com/josephmisiti/awesome-machine-learning

    3. Tự học machine learning trong 100 ngày

    Repo này cho bạn lộ trình, kế hoạch và cách chinh phục machine learning trong vòng 100 ngày.

    https://github.com/Avik-Jain/100-Days-Of-ML-Code

    4. Tổng hợp tất cả các tài liệu hay về NLP – Awesome-nlp

    Tổng hợp tất cả các tài liệu, công cụ awesome nhất về xử lý ngôn ngữ tự nhiên.

    https://github.com/keon/awesome-nlp

    5. Tổng hợp các tài liệu để học Python – Awesome-python

    Mọi tài liệu hay ho để học lập trình python đều có ở đây.

    https://github.com/vinta/awesome-python

    6. Khóa học Tensorflow cơ bản sử dụng code minh họa

    Loạt ví dụ sử dụng tensorflow để giải quyết các bài toán từ cơ bản tới nâng cao. Khóa học tensorflow của tôi cũng có sử dụng nguồn tài liệu này.

    https://github.com/aymericdamien/TensorFlow-Examples

    7. Một kế hoạch học tập hoàn chỉnh để trở thành 1 kỹ sư machine learning

    Tài liệu, các kiến thức cần học để có thể trở thành một kỹ sư machine learning.

    https://github.com/ZuzooVn/machine-learning-for-software-engineers

    8. Danh sách các bộ datasets

    Dữ liệu là không thể thiếu đối với máy học. Repo này chia sẻ cho bạn các bộ dữ liệu được public, hãy ấn Ctrl + F để tìm kiếm. Các datasets trong này có cập nhật mới liên tục.

    https://github.com/awesomedata/awesome-public-datasets

    9. Danh sách các tài nguyên deeplearning

    Tổng hợp tất cả các tài liệu, công cụ cho deep learning.

    https://github.com/endymecy/awesome-deeplearning-resources

    10. Danh sách các bài báo khoa học về deep learning có mô tả

    Danh sách các bài báo, code nếu có về deep learning được cập nhật liên tục. Bạn chỉ cần Ctrl + F để tìm kiếm cái bạn cần mà thôi.

    https://github.com/dennybritz/deeplearning-papernotes

    Danh sách các mã khóa học của ĐH StanFord

    Với các mã lớp học có liên quan về machine learning dưới đây. Bạn chỉ cần tìm kiếm google với từ khóa là mã lớp học, cộng thêm chữ stanford. Bạn sẽ có thể truy cập vào slide bài giảng, bài tập.

    Đối với video bài giảng, bạn hãy lên youtube và tìm kiếm theo cách tương tự.

    • Khóa học về xác suất thống kê(Probability): CS109
    • Khóa học xử lý dữ liệu lớn: CS246
    • Khoá dạy về machine learning cho computer vision: CS231N
    • Khoá dạy về machine learning cho natural language processing: CS224N
    • Khoá phân tích mạng lưới (analysis of network): CS224W
    • Khóa học Tensorflow: CS20SI

    Một số tài liệu machine learning cơ bản tiếng Việt

    Tài liệu tiếng việt trong lĩnh vực này thực sự rất ít. Ngoài blog machinelearningcoban.com của anh Tiệp ra thì mình cũng có một vài link hữu ích bổ sung dưới đây:

    1/ Khóa học Python dành cho người học Machine learning

    2/ Khóa học OpenCV cơ bản của anh Võ Vĩ Hùng

    3/ Khóa học Tensorflow cơ bản của mình và của anh Phạm Văn Toàn

    4/ Sách DeepLearning phiên bản tiếng Việt

    Trên đây là danh sách tài liệu machine learning cơ bản cho người mới bắt đầu. Hi vọng bài chia sẻ này giúp các bạn có những tài liệu học tập bổ ích.

    Chúc các bạn học tập tốt!
